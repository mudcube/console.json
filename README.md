**console.json()**

*JSON syntax highlighting in the browser console :)*

To use simply include in your project and type into the console:

<pre>
console.json({
	'string': 'testing',
	'boolean': true,
	'function': function() {},
	'number': 3123,
	'null': null,
	'undefined': undefined,
	'array': ['a', 'b', 'c'],
	'object': {
		'key': 'value'
	}
});
</pre>
